# homebrew-whatbattery

Homebrew tap for [WhatBattery](https://github.com/darrylmorley/whatbattery), a
menu bar app for battery health and live power on your Mac, iPhone, and iPad.

## Install

```sh
brew install --cask darrylmorley/whatbattery/whatbattery
```

The cask's `version` and `sha256` are bumped automatically by the WhatBattery
release script on each release.
